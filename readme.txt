基于神经网络的格子玻尔兹曼方法顶盖驱动流湍流大涡模拟程序使用文档
一．环境配置
python环境：anaconda3
软件包：numpy1.24.3，numba0.57.1，panda2.0.3，openpxyl3.0.10
TensorflowCPU，matplotlib3.7.2，scikit-learn1.3.0
二．使用说明
使用LBM.py程序，设置格子大小如1024*1024，雷诺数，文件写入名，终止迭代次数，或者误差值。运行，得到该雷诺数的训练集。
设置读入文件名和输出文件名，使用delete.py程序去除训练集中的无意义数据。
使用11.py程序，调整分割操作，要训练多个神经网络，分割数据的输入有多个，可以调节为带坐标的也就是x，y坐标，和不带坐标的，分割数据的输出只有一个，训练神经网络，注意注释中的操作为归一化操作可以选择使用或者不使用。
使用训练好的神经网络在ANN-LES.py程序中读入，运行得到流场数据
使用tecplot软件读取流场数据得到流场图。
